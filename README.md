Static sites template with Jade, Scss and Gulp.js
=================================================

The template includes:

- Jade
- Plain HTML
- Sass (3.3)
- Bourbon/Neat/Bitters
- Autoprefixer
- BrowserSync
- Gulp.js
- Nanoscroller
- BXSlider

To get started:

1. `git clone https://github.com/CosminAnca/static-gulp-jade.git [project-name]`
2. `cd [project-name]`
3. `npm install`
4. `gulp`
5. Done
